# Task: Badge Size Radio Boxes

## Description
Replace the badge size slider with fancy, usable radio boxes for Small, Medium, and Large options. Update the badge rendering to use fixed pixel sizes based on the 1024x1024 final output.

## Acceptance Criteria
- [ ] Replace slider with radio box selection
- [ ] Three options: Small, Medium, Large
- [ ] Visual radio boxes (not plain inputs)
- [ ] Real-time preview updates when size changes
- [ ] Accessible (keyboard navigation, ARIA labels)
- [ ] Works in both light and dark themes

## Technical Details

### Badge Sizes (for 1024x1024 output)

| Size | Dimensions | Percentage |
|------|------------|------------|
| Small | 150 x 150 px | ~14.6% |
| Medium | 225 x 225 px | ~22% |
| Large | 350 x 350 px | ~34% |

### Type Definition
```typescript
type BadgeSize = 'small' | 'medium' | 'large'

interface BadgeSizeOption {
  value: BadgeSize
  label: string
  pixels: number
  description: string
}

const badgeSizeOptions: BadgeSizeOption[] = [
  { value: 'small', label: 'S', pixels: 150, description: 'Small (150px)' },
  { value: 'medium', label: 'M', pixels: 225, description: 'Medium (225px)' },
  { value: 'large', label: 'L', pixels: 350, description: 'Large (350px)' }
]
```

### Component Implementation
```vue
<template>
  <div class="badge-size-selector" role="radiogroup" aria-label="Badge size">
    <button
      v-for="option in sizeOptions"
      :key="option.value"
      role="radio"
      :aria-checked="selectedSize === option.value"
      :class="['size-option', { 'size-option--selected': selectedSize === option.value }]"
      @click="selectSize(option.value)"
    >
      <span class="size-letter">{{ option.label }}</span>
      <span class="size-name">{{ option.value }}</span>
    </button>
  </div>
</template>
```

### Update useImageProcessor
```typescript
const BADGE_SIZES = {
  small: 150,
  medium: 225,
  large: 350
} as const

type BadgeSize = keyof typeof BADGE_SIZES

const badgeSize = ref<BadgeSize>('medium')

function getBadgePixelSize(): number {
  return BADGE_SIZES[badgeSize.value]
}
```
