# Task: Testing and Deployment

## Description
Perform comprehensive cross-browser testing, run Lighthouse audits, fix any issues, and deploy to Vercel. Ensure the app meets all performance and quality targets.

## Acceptance Criteria
- [ ] Works on Chrome (desktop + mobile)
- [ ] Works on Firefox (desktop)
- [ ] Works on Safari (desktop + iOS)
- [ ] Works on Edge (desktop)
- [ ] Lighthouse Performance score > 90
- [ ] Lighthouse Accessibility score > 90
- [ ] No console errors in production
- [ ] Deployed to Vercel with custom domain (optional)
- [ ] All error scenarios handled gracefully

## Technical Details

### Browser Testing Matrix
| Browser | Platform | Priority |
|---------|----------|----------|
| Chrome | Desktop | High |
| Chrome | Android | High |
| Safari | Desktop | High |
| Safari | iOS | High |
| Firefox | Desktop | Medium |
| Edge | Desktop | Medium |

### Test Scenarios
1. **Upload Flow**
   - Upload valid PNG (< 5MB)
   - Upload valid JPG (< 5MB)
   - Attempt upload > 5MB (expect error)
   - Attempt upload invalid format (expect error)

2. **Crop Flow**
   - Pan crop area
   - Zoom in/out
   - Confirm crop

3. **Badge Flow**
   - Select badge
   - Change position
   - Remove badge
   - Select different badge

4. **Download Flow**
   - Download with badge
   - Download without badge (if allowed)
   - Verify 1024x1024 output

5. **Reset Flow**
   - Start Over with confirmation
   - Cancel confirmation
   - Confirm reset

### Lighthouse Audit
```bash
# Run Lighthouse CI
npx lighthouse https://badge-it.vercel.app --output html --output-path ./lighthouse-report.html
```

### Performance Targets
| Metric | Target |
|--------|--------|
| First Contentful Paint | < 1.5s |
| Largest Contentful Paint | < 2.5s |
| Time to Interactive | < 3.0s |
| Cumulative Layout Shift | < 0.1 |
| Total Blocking Time | < 200ms |

### Vercel Deployment
```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel

# Production deploy
vercel --prod
```

### Vercel Configuration (vercel.json)
```json
{
  "buildCommand": "nuxt build",
  "outputDirectory": ".output/public",
  "framework": "nuxtjs"
}
```
