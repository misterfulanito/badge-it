# Task: Stepper Visual Fixes

## Description
Fix visual issues with the horizontal stepper component: misaligned connector lines, remove interactivity, and center-align step titles.

## Acceptance Criteria
- [ ] Connector lines properly positioned between circles (not overlaying)
- [ ] Steps are non-interactive (no hover effects, no click handlers)
- [ ] Step titles/labels are center-aligned under each circle
- [ ] Clean, professional appearance
- [ ] Reduced to 3 steps (Upload & Crop, Badge, Preview)

## Technical Details

### Current Issues
1. Horizontal lines overlay the step circles instead of connecting them
2. Steps have hover effects and are clickable
3. Titles may not be centered

### Fixes Required

#### 1. Connector Line Alignment
```css
/* Lines should connect circles, not overlay them */
.step-connector {
  position: absolute;
  top: 50%;
  left: calc(50% + 16px); /* Start after circle */
  width: calc(100% - 32px); /* Width minus both circle radii */
  height: 2px;
  transform: translateY(-50%);
}
```

#### 2. Remove Interactivity
```css
.step {
  cursor: default; /* Not pointer */
  pointer-events: none; /* Disable all interactions */
}

.step:hover {
  /* Remove all hover effects */
  transform: none;
  background-color: inherit;
}
```

#### 3. Center-Align Labels
```css
.step-label {
  text-align: center;
  width: 100%;
}
```
