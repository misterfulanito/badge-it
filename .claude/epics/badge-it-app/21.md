# Task: Replace Crop Library

## Description
Replace vue-advanced-cropper with a Vue-compatible version of react-easy-crop. Since react-easy-crop is React-specific, use vue-easy-cropper or implement a similar cropping experience.

## Acceptance Criteria
- [ ] Remove vue-advanced-cropper dependency
- [ ] Install and configure vue-easy-cropper (or equivalent)
- [ ] Maintain 1:1 aspect ratio for cropping
- [ ] Zoom functionality (50% - 200%)
- [ ] Pan/drag to position crop area
- [ ] Smooth, responsive cropping experience
- [ ] Best image quality maintained
- [ ] Works on mobile (touch support)

## Technical Details

### Library Options

**Option 1: vue-easy-cropper**
```bash
npm install vue-easy-cropper
```

**Option 2: vue3-cropper**
```bash
npm install vue3-cropper
```

**Option 3: Custom implementation using Cropper.js**
```bash
npm install cropperjs
```

### Required Features
1. Fixed 1:1 aspect ratio (square crop)
2. Zoom via slider (50% to 200%)
3. Drag to reposition
4. Touch support for mobile
5. High resolution output

### Cropper Configuration
```typescript
const cropperOptions = {
  aspectRatio: 1, // 1:1 square
  viewMode: 1,    // Restrict crop box to canvas
  dragMode: 'move',
  autoCropArea: 1,
  responsive: true,
  guides: false,
  center: true,
  highlight: false,
  cropBoxMovable: false,
  cropBoxResizable: false,
  toggleDragModeOnDblclick: false,
}
```

### Zoom Mapping
```typescript
const MIN_ZOOM = 0.5  // 50%
const MAX_ZOOM = 2.0  // 200%
const ZOOM_STEP = 0.25 // 25% increments

// Convert slider percentage to zoom factor
function percentToZoom(percent: number): number {
  return percent / 100
}
```

### Output Quality
- Maintain original image resolution during cropping
- Use canvas with proper pixel ratio for crisp output
- Final export: 1024x1024px PNG

### Migration Steps
1. Uninstall vue-advanced-cropper
2. Install new cropping library
3. Update ImageCropper.vue component
4. Update any type definitions
5. Test thoroughly on desktop and mobile
