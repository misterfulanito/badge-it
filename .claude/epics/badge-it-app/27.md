# Task: Create Single-Page Two-Column Layout

## Description
Restructure the main page to a two-column layout where left column handles upload/preview and right column handles badge options.

## Layout Specification
```
┌─────────────────────────────────────────────────────────┐
│  Instructional Copy (centered, above columns)           │
│  "Upload your profile image for Slack or any social     │
│   network, then add a badge to make it stand out."      │
├────────────────────────┬────────────────────────────────┤
│   Left Column (60%)    │   Right Column (40%)           │
│   Upload/Preview Area  │   Badge Options                │
│   1:1 aspect ratio     │   (with disabled state)        │
├────────────────────────┴────────────────────────────────┤
│  Sticky Bottom Bar [Share] [Preview] [Download]         │
└─────────────────────────────────────────────────────────┘
```

## Acceptance Criteria
- [ ] Two-column grid: 60% left, 40% right
- [ ] 32px gap between columns
- [ ] Responsive: stack on mobile (<768px)
- [ ] Left column maintains 1:1 aspect ratio
- [ ] Remove step headers (number circle, title, subtitle)
- [ ] Add instructional copy above columns

## CSS Grid Structure
```css
.main-layout {
  display: grid;
  grid-template-columns: 3fr 2fr;
  gap: 32px;
}
```
