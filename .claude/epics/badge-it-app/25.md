# Epic: Single Page Redesign

## Overview

Consolidate the multi-step badge creation flow into a single-page experience. Remove the stepper navigation and simplify the UX to: upload image → configure badge → download.

## Goals

1. **Simplify UX** - One page, no navigation between steps
2. **Reduce complexity** - Remove unused code, libraries, components
3. **Improve usability** - Clear visual hierarchy, disabled states, helpful copy
4. **Add features** - Preview modal, share functionality
5. **Maintain quality** - Best image quality throughout

## Architecture

### Layout Structure
```
┌─────────────────────────────────────────────────────────┐
│  Header (Logo + Theme Toggle)                           │
├─────────────────────────────────────────────────────────┤
│  Instructional Copy                                     │
├────────────────────────┬────────────────────────────────┤
│                        │                                │
│   Left Column (60%)    │   Right Column (40%)           │
│   - Upload Area        │   - Badge Selection            │
│   - Image Preview      │   - Position Options           │
│   - Crop Controls      │   - Size Options (S/M/L)       │
│   - Change/Remove      │   (Disabled until upload)      │
│                        │                                │
├────────────────────────┴────────────────────────────────┤
│  Sticky Bottom Bar                                      │
│  [Share] [Preview] [Download]                           │
├─────────────────────────────────────────────────────────┤
│  Footer (Privacy Note + Links)                          │
└─────────────────────────────────────────────────────────┘
```
