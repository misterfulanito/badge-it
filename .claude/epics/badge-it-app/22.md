# Task: Badge Step Layout Fix

## Description
Fix the Badge step two-column layout with proper proportions: 60% left column (preview), 40% right column (options), 32px gap between columns. Fully responsive with single column on mobile.

## Acceptance Criteria
- [ ] Left column: 60% width with image preview
- [ ] Right column: 40% width with options
- [ ] 32px margin/gap between columns
- [ ] Image adapts to fill left column width
- [ ] Accordion components fit within right column (no overflow)
- [ ] Responsive: single column on mobile
- [ ] Flexible components that adapt to container size
- [ ] Works on all screen sizes (small to large)

## Technical Details

### Desktop Layout (min-width: 768px)
```
┌────────────────────────────┬─────────────────────┐
│                            │                     │
│     Preview Image          │   Badge Options     │
│     (60% width)            │   (40% width)       │
│                            │                     │
│     [Image fills width]    │   [Accordions]      │
│                            │                     │
│                            │   - Select Badge    │
│                            │   - Position        │
│                            │   - Size            │
│                            │                     │
└────────────────────────────┴─────────────────────┘
          ← 32px gap →
```

### Mobile Layout (max-width: 767px)
```
┌─────────────────────────────┐
│      Preview Image          │
│      (100% width)           │
└─────────────────────────────┘
┌─────────────────────────────┐
│      Badge Options          │
│      (100% width)           │
│                             │
│      - Select Badge         │
│      - Position             │
│      - Size                 │
└─────────────────────────────┘
```

### CSS Implementation
```css
.badge-step-layout {
  display: grid;
  grid-template-columns: 1fr;
  gap: 1.5rem;
}

@media (min-width: 768px) {
  .badge-step-layout {
    grid-template-columns: 60% 40%;
    gap: 32px; /* 2rem */
  }
}

/* Left column - Preview */
.preview-column {
  width: 100%;
}

.preview-column .preview-image {
  width: 100%;
  height: auto;
  aspect-ratio: 1;
}

/* Right column - Options */
.options-column {
  width: 100%;
  max-width: 100%; /* Prevent overflow */
}

.options-column .collapsible-section {
  width: 100%;
  max-width: 100%;
}
```

### Flexbox Alternative
```css
.badge-step-layout {
  display: flex;
  flex-wrap: wrap;
  gap: 32px;
}

.preview-column {
  flex: 1 1 60%;
  min-width: 280px;
}

.options-column {
  flex: 1 1 calc(40% - 32px);
  min-width: 280px;
}
```
