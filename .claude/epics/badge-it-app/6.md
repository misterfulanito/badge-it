# Task: Badge Positioning Component

## Description
Create the BadgePositioner component using PrimeVue SelectButton to allow users to choose badge placement: left-bottom, center-bottom, or right-bottom. Default position is center-bottom.

## Acceptance Criteria
- [ ] Three position options displayed as buttons
- [ ] Visual icons or labels for each position (Left, Center, Right)
- [ ] Only one position can be selected at a time
- [ ] Default selection is center-bottom
- [ ] Selected position has clear visual indicator
- [ ] Emits position value on change
- [ ] Disabled state when no badge is selected

## Technical Details
- **Component**: `components/BadgePositioner.vue`
- **PrimeVue Component**: SelectButton

### Position Values
```typescript
type BadgePosition = 'left' | 'center' | 'right'

const positions = [
  { label: 'Left', value: 'left', icon: 'pi pi-align-left' },
  { label: 'Center', value: 'center', icon: 'pi pi-align-center' },
  { label: 'Right', value: 'right', icon: 'pi pi-align-right' }
]
```

### Component Structure
```vue
<template>
  <div class="badge-positioner">
    <label>Badge Position</label>
    <SelectButton
      v-model="position"
      :options="positions"
      optionLabel="label"
      optionValue="value"
      :disabled="!hasBadgeSelected"
      @change="$emit('update:position', position)"
    />
  </div>
</template>
```

### Position Mapping for Canvas
```typescript
// Used by PreviewCanvas to calculate badge coordinates
const getPositionCoordinates = (
  position: BadgePosition,
  canvasSize: number,
  badgeSize: number
): { x: number; y: number } => {
  const y = canvasSize - badgeSize - padding // Bottom position

  switch (position) {
    case 'left':
      return { x: padding, y }
    case 'center':
      return { x: (canvasSize - badgeSize) / 2, y }
    case 'right':
      return { x: canvasSize - badgeSize - padding, y }
  }
}
```
