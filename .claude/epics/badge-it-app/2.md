# Task: Project Setup

## Description
Initialize a new Nuxt 3 project with PrimeVue 4 UI library and all required dependencies. Set up the project structure, configure PrimeVue theming, and create the basic layout scaffolding.

## Acceptance Criteria
- [ ] Nuxt 3 project initialized with TypeScript
- [ ] PrimeVue 4 installed and configured
- [ ] PrimeVue theme configured (Aura or Lara theme)
- [ ] Project structure matches architecture in epic
- [ ] Dev server runs without errors
- [ ] Basic app.vue and pages/index.vue created
- [ ] ESLint and Prettier configured

## Technical Details
- **Framework**: Nuxt 3 with TypeScript
- **UI Library**: PrimeVue 4 with @primevue/themes
- **Additional packages**: vue-advanced-cropper

### Installation Commands
```bash
npx nuxi@latest init badge-it
cd badge-it
npm install primevue @primevue/themes
npm install vue-advanced-cropper
```

### nuxt.config.ts Setup
```typescript
export default defineNuxtConfig({
  modules: ['@primevue/nuxt-module'],
  primevue: {
    options: {
      theme: {
        preset: Aura
      }
    }
  },
  css: ['~/assets/styles/main.css']
})
```
