# Task: Image Cropper Component

## Description
Create the ImageCropper component using vue-advanced-cropper with a locked 1:1 aspect ratio. Users can adjust the crop area position and zoom level. The component displays at 512x512px on desktop and responsive width on mobile.

## Acceptance Criteria
- [ ] Cropper displays uploaded image
- [ ] Aspect ratio locked to 1:1 (square)
- [ ] User can drag to reposition crop area
- [ ] User can zoom in/out
- [ ] Preview canvas is 512x512px on desktop
- [ ] Preview canvas is 100% width on mobile (maintains 1:1 ratio)
- [ ] Emits cropped image data when user confirms
- [ ] Smooth, responsive interactions (60fps)

## Technical Details
- **Component**: `components/ImageCropper.vue`
- **Library**: vue-advanced-cropper
- **Output**: Cropped ImageData or canvas

### Component Structure
```vue
<template>
  <div class="cropper-container">
    <Cropper
      ref="cropperRef"
      :src="imageSrc"
      :stencil-props="{
        aspectRatio: 1
      }"
      :resize-image="{
        adjustStencil: false
      }"
      @change="onCropChange"
    />
  </div>
</template>
```

### Responsive Sizing
```css
.cropper-container {
  width: 512px;
  height: 512px;
}

@media (max-width: 768px) {
  .cropper-container {
    width: 100%;
    height: auto;
    aspect-ratio: 1;
  }
}
```

### Getting Cropped Result
```typescript
const getCroppedImage = () => {
  const { canvas } = cropperRef.value.getResult()
  return canvas // Returns HTMLCanvasElement
}
```
