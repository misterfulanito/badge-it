# Task: Add Share Button with Social Options

## Description
Add a share button in the bottom bar that allows users to share the Badge It page via various social platforms.

## Social Platforms
1. **Email** - mailto: link with subject and body
2. **WhatsApp** - wa.me share URL
3. **Twitter/X** - twitter.com/intent/tweet
4. **Instagram** - Copy link (IG doesn't support direct share)
5. **LinkedIn** - linkedin.com/sharing/share-offsite

## Acceptance Criteria
- [ ] Share button in bottom action bar
- [ ] Dropdown/popover with social options
- [ ] Each platform opens in new tab/window
- [ ] Instagram shows "Copy Link" with toast confirmation
- [ ] Icons for each platform
- [ ] Share URL is the app URL (not the image)
- [ ] Pre-filled share text: "Create badges for your profile pictures with Badge It!"

## Share URLs
```typescript
const SHARE_TEXT = "Create badges for your profile pictures with Badge It!"
const APP_URL = window.location.href

const shareLinks = {
  email: `mailto:?subject=Check out Badge It&body=${SHARE_TEXT} ${APP_URL}`,
  whatsapp: `https://wa.me/?text=${encodeURIComponent(SHARE_TEXT + ' ' + APP_URL)}`,
  twitter: `https://twitter.com/intent/tweet?text=${encodeURIComponent(SHARE_TEXT)}&url=${encodeURIComponent(APP_URL)}`,
  linkedin: `https://www.linkedin.com/sharing/share-offsite/?url=${encodeURIComponent(APP_URL)}`
}
```

Create `components/ShareButton.vue` or integrate into page
