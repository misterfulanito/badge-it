# Task: Add Preview Modal Functionality

## Description
Add a preview button in the bottom bar that opens a modal showing the final image at full size (proportioned to max viewport size).

## Acceptance Criteria
- [ ] Preview button in bottom action bar
- [ ] Button disabled until image is uploaded
- [ ] Modal opens with full-size preview
- [ ] Image maintains aspect ratio (1:1)
- [ ] Image sized to fit viewport (max 90vh or 90vw)
- [ ] Close button (X) and click-outside to close
- [ ] Keyboard accessible (Escape to close)
- [ ] Shows badge overlay on preview

## Modal Design
```
┌──────────────────────────────────────┐
│                              [X]     │
│                                      │
│         ┌──────────────┐             │
│         │              │             │
│         │   Preview    │             │

│         │   1:1        │             │
│         │              │             │
│         └──────────────┘             │
│                                      │
│     "Your image at full resolution"  │
└──────────────────────────────────────┘
```

Create `components/PreviewModal.vue`:
- Uses PrimeVue Dialog or custom modal
- Receives canvas/image as prop
- Full-screen overlay with centered content
